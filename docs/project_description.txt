1. fintune: 학습 코드입니다. 이 코드는 학습 데이터(fintune_dataset)를 사용하여 모델의 가중치를 훈련하고, 리더보드 점수와 동일한 결과를 내는 모델 파일을 생성할 수 있습니다.

2. fintune_dataset: 학습 데이터셋입니다. 모델 훈련에 사용된 모든 데이터 파일이 포함됩니다. 코드를 실행할 때 이 데이터셋을 불러와야 합니다. fintune 폴더 안에 내장 되어있습니다. 각 데이터의 라이센스도 csv형태로 저장되어 있습니다.

3. gototop: 추론 코드입니다. 훈련된 모델 가중치와 함께 사용되어 리더보드 제출 시와 동일한 추론 결과를 생성하는 코드입니다. 이 코드는 fintune 코드가 생성한 모델 가중치 파일을 사용하여 추론을 실행합니다.

4. condalist.txt: 환경 정보 파일입니다. fintune 코드를 실행했을 때 사용된 가상 환경의 라이브러리 목록(conda list)과 GPU 정보를 담고 있습니다. 
----
각 모델의 라이센스는 script.py 파일 상단 각주에 표시되어있습니다.
